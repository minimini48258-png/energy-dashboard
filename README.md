# ⚡ 電力需給分析ダッシュボード

公共施設の30分値電力使用量データを可視化・分析するStreamlitアプリです。
地域新電力事業のエネルギー小売検討に活用することを想定しています。

## 機能（MVP）

- **Excelデータ読み込み**：複数ファイル対応、列名の自動マッピング
- **需要カーブ表示**：施設別 / 全施設合計の30分値折れ線グラフ
- **KPI表示**：積算使用量・最大・平均・最小
- **需要パターン分析**：
  - 月別使用量
  - 時間帯別平均使用量
  - 平日・休日比較
  - 施設別年間使用量ランキング
  - 日別使用量
- **需給バランス**：太陽光・蓄電池・市場調達との需給比較
- **収支シミュレーション**：小売単価一律での簡易P&L試算
- **小売FS（詳細収支）**：
  - 施設ごとの契約電力・電圧区分・料金プラン（基本料金＋従量料金：一律／時間帯別）
  - 電圧区分別の託送料金、燃料費調整、再エネ賦課金、容量拠出金（簡易試算）
  - JEPX想定単価・予備費率を反映した電力調達費
  - 損益計算書ふうの月別・年間サマリー、JEPX価格の感度分析、CO2排出量・地産電源比率
  - 「電力小売FS 収支試算ツール」の計算ロジック（売上総利益 = 売上高 − 再エネ賦課金 −
    (電力調達費＋託送料金＋容量拠出金)）に準拠した簡易版

## 対応データ形式

| 列名 | 内容 |
|------|------|
| `datetime` | 日時（30分刻み）|
| `facility_name` | 施設名 |
| `consumption_kwh` | 使用電力量 (kWh/30min) |

列名が異なる場合は読み込み後に手動マッピングできます。
日本語列名（`日時`、`施設名`、`使用電力量`など）にも対応しています。

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動

```bash
streamlit run app.py
```

## サンプルデータの生成

```bash
python generate_sample.py
```

5施設・1年分（87,600行）のサンプルCSVとExcelが `data/sample/` に生成されます。

## ファイル構成

```
energy-dashboard/
├── app.py              # Streamlitメインアプリ
├── data_loader.py      # Excel/CSV読み込み・列マッピング
├── data_cleaner.py     # データクレンジング・品質チェック
├── analyzer.py         # 集計・分析ロジック
├── visualizer.py       # Plotlyグラフ生成
├── financial_model.py  # 簡易収支シミュレーション（小売単価一律）
├── retail_fs.py        # 小売FS：料金プラン・託送・容量拠出金・CO2の詳細計算
├── supply_planner.py   # 電源登録・供給プロファイル生成
├── generate_sample.py  # サンプルデータ生成
├── requirements.txt
└── data/sample/        # サンプルデータ（.gitignoreでExcel除外）
```

## 小売FSタブの注意事項

料金プラン・託送料金・再エネ賦課金・容量拠出金の初期値はいずれも目安値です。
実際の数値は各エリアの料金表・託送供給等約款・OCCTO公表資料等で確認のうえ入力してください。
施設設定・料金プランはセッションを超えて `/tmp/energy_dashboard/` にJSON保存されます
（同一環境での再利用を想定。異なる環境へ複製する場合は該当ファイルをコピーしてください）。
